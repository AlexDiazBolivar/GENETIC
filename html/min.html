genetic.min

Main interface for minimisation.

Syntax

[xopt, fopt, info] = genetic.min(f, x, method)
[xopt, fopt, info] = genetic.min(f, x, method, options)
[xopt, fopt, info] = genetic.min(f, x, method, constraints, options)

Arguments

Inputs

f The objective function. More specifically f can either be
  • a handle function,
  • a cell containing a handle function and the number of objectives, i.e. f = {f_handle, fDim} (mandatory for multi-objective optimisation),
  • a string with the name of a benchmark available in genetic.
x Dimension of the problem or and initial point/population.
method Name of the method to be used.
options A structure containing the parameters for the optimisation process.
constraints A structure containing a description of the constraints of the optimisation problem.

Outputs

xopt Best solution(s) found.
fopt Associated objective of Pareto front.
info Structure containing informations about the optimisation process.
